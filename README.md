# Tiger-Crew-Server | Tiger服务器

A Minecraft PE (Bedrock) Server created by some students.

一个由学生建设的Mincraft基岩版服务器。

## Creation | 创建

This project was created in November 2020.

本服务器在2020年11月创建。

## Builders | 主要建造者

Harry Wang; Boyang Li; Scott Zhong
